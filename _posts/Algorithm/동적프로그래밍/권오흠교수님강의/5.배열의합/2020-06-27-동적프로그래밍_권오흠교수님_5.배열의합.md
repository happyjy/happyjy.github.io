# 문제
  * 배열의 숫자 요소를 모두 더하기

# 해결 방법
  * arr[0] ... arr[n-1] 까지의 합을 구하여 반환

# CODE
  ``` js
    function sum(n, arr) {
      if (n <= 0) {
        return 0;
      } else {
        return sum(n - 1, arr) + arr[n - 1];
      }
    }

    var arr = [1, 2, 3, 4, 5];
    console.log(sum(5, arr));
  ```
