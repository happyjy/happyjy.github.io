# 프로토콜이란?

컴퓨터, 단말기 사이에서 데이터 통신을 원활하게 하기 위해서 필요한 통신규약, 신호송신의 순서, 데이터 표현법

## 프로토콜의 기능

- 캡슐화(encapsulation)
- 흐름제어(flow control)
- 연결제어(connection control)
- 주소설정(addressing)
- 순서설정(sequence)
- 동기화(synchronization)
- 세분화와 재합성(fragmentation and reassembly)
- 다중화(multiplexing)
